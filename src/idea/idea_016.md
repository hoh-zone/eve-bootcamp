# 16. 去中心化采矿订单板 (Alliance Job Board)

## 💡 核心概念 (Concept)
类似于外包发布平台。任何人发布所需的资源（如：需要 1000 万吨冰矿），存入 SUI 悬赏。任何散人只要把矿存入指定储物箱，合约自动结账。

## 🧩 解决的痛点 (Pain Points Solved)
- **原材料断供**：大联盟忙着打仗，没有时间组织采矿。
- **零工信任门槛**：散人矿工不敢给大公会干活，担心被赖账。

## 🎮 详细玩法与机制 (Gameplay Mechanics)
1. **履约智能锚定**：发单方将悬赏金额和所需的矿石 Object 定义存入合约。
2. **物品存入触发**：利用 EVE World 的储物箱权限钩子。当指定 ID 的矿石对象被放入金库，合约感知并扣减任务配额。
3. **原子结汇**：在一笔交易中完成“矿石所属权转移”和“SUI 酬劳支付”，完全无需管理员干预。
