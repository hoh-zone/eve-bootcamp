# 20. 全宇宙全民基本收入发生器 (UBI Generator)

## 💡 核心概念 (Concept)
一个完全公益的 DAO 金库，每天根据全服交易税或大佬捐赠，按比例向注册时间不满 7 天或资产低于特定阈值的新手自动空投低保物资。

## 🧩 解决的痛点 (Pain Points Solved)
- **新手流失率高**：开局被老玩家洗劫一空，无法继续采矿和跃迁。
- **生态多样性差**：贫富差距过大，导致游戏后期变成死忠大佬的小圈子。

## 🎮 详细玩法与机制 (Gameplay Mechanics)
1. **身份认证与防刷**：通过 SuiNS 注册时长或特定的防刷认证（如 zkLogin 关联的非临时邮箱）来核实新手身份。
2. **流动能量池**：全服所有的商业设施（星门、收费站）提取 0.1% 的利润流向此 UBI 合约。
3. **每日一键申领**：新手玩家点击 UI 即可通过 PTB 同时领走一小捆燃料、几颗普通炮弹和少量 SUI Gas 补贴。

## 🛠️ Sui 核心特性应用 (Sui Features)
- [x] Sponsored Transactions (免除新手领低保时的 Gas 费)
- [x] zkLogin (社交身份绑定校验)

## 📐 智能合约架构规划 (Smart Contract Architecture)
### 核心 Object
- `UBITreasury`: 存储资助金的共享对象。
- `ClaimLog`: 防止重复领取的历史表。
